Visual Epicure will be a Progressive Web Application (PWA) built using Python / Flask allowing users to:

- Freely draw and visualise
- Use tools likes insert text, draw straight lines, insert objects and images
- Easy ways to make brainstorms and mood boards
- Create multiple whiteboards with unlimited/limited boundaries


Key Features:
- Functional
	- User Authentication, dashboard, customisable folders
- Performance
	- Fast
- Security
	- 
- Privacy
	- Compliance with the Australian Privacy Act
